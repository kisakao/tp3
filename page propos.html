Résumé

Le projet qui suit est une maquette pour un site sur le jeu Genshin Impact. Il agit comme un site wiki qui donne des informations de base ainsi que des données sur les personnages (pour des raisons de format, nous n’avons pas pu mettre tous les personnages). Nous avons ajouté une section ou les utilisateurs peuvent demander des questions.

Avis sur notre projet

Avant de produire notre maquette, nous avons visité des sites pour comprendre la compétition et aussi pour apprendre les différents aspects que nous devrons ajouter dans notre propre site. Nous avons alors décidé de mettre dans notre site le font de texte ainsi que des icônes retrouvables dans le jeu Genshin Impact. Le but derrière cette décision est que l’insertion de ces aspects dans notre site web pour donner un sens de familiarité pour les joueurs du jeu et pour les gens ayant aucune idée du jeu. Pour les connaisseurs de Genshin Impact, les additions que nous avons faites seront reconnu par eux ce qui, nous présumons, aura un effet positif par rapport à leur expérience du site. Pour les moins connaissants, le fait qu’ils verront des choses retrouvables dans le jeu, surtout nos icônes, aidera leur expérience quand ils joueront le jeu pour la première fois.
